# Day 4 - Object Oriented Programming (OOP) Assignment

## 📌 Project Overview

This repository contains Day 4 OOP assignments based on fundamental Object Oriented Programming concepts in Python.

The objective of this assignment is to understand how classes and objects help us build reusable, organized and scalable programs.

All programs are implemented using beginner-friendly Python code suitable for a 2nd year B.Tech Computer Science Engineering student.

---

## 🎯 Learning Objectives

By completing this assignment, we can learn:

- Create classes and objects
- Use constructors (__init__)
- Work with instance variables and methods
- Understand class variables
- Implement encapsulation
- Implement inheritance
- Demonstrate polymorphism
- Apply abstraction
- Override methods
- Use super() function
- Understand access modifiers
